# Adidas Sales Performance Analysis 📊

## 📝 Project Overview
This project presents a detailed analysis of **Adidas Sales Data (2020-2021)**. The goal was to identify key performance drivers, regional sales trends, and profitability across different product categories and retailers.

## 🚀 Key Insights
*   **Total Sales:** $899.9M
*   **Total Profit:** $332.1M (Average Margin: 37%)
*   **Best Category:** Men's Street Footwear
*   **Top Region:** West Region
